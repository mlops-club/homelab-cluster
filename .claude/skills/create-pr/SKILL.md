# create-pr skill

## Instructions

When the user asks to create a PR, open a PR, submit a PR, or push for review:

1. Read `.ai/howto/how-to-create-a-pr.md` for the full procedure
2. Read `.ai/templates/pr-body.md.template` for the PR body format
3. Follow the steps in the howto guide

## Allowed Tools

- Bash (git and gh commands only)
- Read
- Glob
- Grep

## Restrictions

- Do not modify code files — only create the PR from existing committed changes
- Do not auto-merge
- Never force-push
